# classbot-app
"A beginner-friendly Streamlit chatbot app that helps students manage assignments and due dates ethically and safely."
ClassBot is a simple yet powerful chatbot application built during a 2-Day AI Sprint. Designed using Streamlit, it allows students to:

Track subjects and due dates

Ask assignment-related questions

Get friendly, clear responses

Receive safe refusals to inappropriate or vague questions

Built with a focus on usability, ethical design, and student productivity, ClassBot demonstrates how even beginner AI developers can build interactive apps with responsible behavior and clean UI.
